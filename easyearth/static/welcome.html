ğŸ§© Install EasyEarth Plugin in QGIS

Manual Installation

  1. Open QGIS > Settings > User Profiles > Open Active Profile Folder
  2. Navigate to python/plugins
  3. Copy easyearth_plugin folder
  4. Restart QGIS, enable the plugin via Plugins menu

Terminal Installation

cd ~/Downloads/easyearth_plugin
cp -r ./easyearth_plugin ~/.local/share/QGIS/QGIS3/profiles/default/python/plugins
