EasyEarth: Run Vision(-Language) Models for Earth Observations

Seamlessly apply cutting-edge computer vision and vision-language models directly on Earth observation data — no coding required.

Watch Demo

🔧 Key Components

  • Server-Side Infrastructure – Scalable backend to run AI models on geospatial data
  • QGIS Plugin GUI – User-friendly interface to apply models inside QGIS
  • Model Manager *(in development)* – Upload, version, and deploy models

✅ Get Started

Requirements

  • Docker Compose ≥ 1.21.2 – Install Guide
  • Python ≥ 3.6
  • QGIS
  • CUDA ≥ 12.4 – Download

Clone Repository

cd ~/Downloads
git clone https://github.com/YanCheng-go/easyearth.git
cp -r ./easyearth/easyearth_plugin easyearth_plugin

Run in Local Mode

chmod +x ./launch_server_local.sh
./launch_server_local.sh

Set Up Docker Server

cd easyearth_plugin
chmod +x ./setup.sh
./setup.sh

Stop the Server

cd easyearth_plugin
sudo docker-compose down

Useful Docker Commands

docker ps -a
docker images
docker rm $(docker ps -a -q)
docker rmi $(docker images -q)
docker volume rm $(docker volume ls -q)
sudo docker inspect <container_id>
sudo docker exec -it <container_id> /usr/src/app

🧩 Install EasyEarth Plugin in QGIS

Manual Installation

  1. Open QGIS > Settings > User Profiles > Open Active Profile Folder
  2. Navigate to python/plugins
  3. Copy easyearth_plugin folder
  4. Restart QGIS, enable the plugin via Plugins menu

Terminal Installation

cd ~/Downloads/easyearth_plugin
cp -r ./easyearth_plugin ~/.local/share/QGIS/QGIS3/profiles/default/python/plugins

🚀 Usage

Run in QGIS

  1. Stop external Docker containers: 
    cd easyearth_plugin
sudo docker-compose down
  2. Open QGIS, click **Start Docker**
  3. Load an image using **Browse Image**
  4. Click **Start Drawing**
